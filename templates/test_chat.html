🧪 Test du Chat Bot AYORA

📋 Instructions de test

Cette page permet de tester le chat bot sans interférer avec l'interface principale.

  1. Cliquez sur l'icône 💬 en haut à droite pour ouvrir le chat
  2. Testez les suggestions rapides en bas du chat
  3. Posez des questions au bot
  4. Vérifiez que les réponses sont pertinentes

🔧 Tests techniques

📱 Test responsive

Redimensionnez votre navigateur pour tester l'adaptation mobile du chat.

Le chat devrait s'adapter automatiquement à la taille de l'écran.

🎨 Test thème

Changez de thème dans les paramètres pour voir l'adaptation du chat.

Le chat devrait s'adapter automatiquement au thème clair/sombre.

✅ Checklist de validation

  • ❌ Bouton chat visible dans le header
  • ❌ Chat s'ouvre au clic
  • ❌ Interface chat complète
  • ❌ Suggestions rapides fonctionnelles
  • ❌ Envoi de messages
  • ❌ Réponses du bot
  • ❌ Fermeture du chat
  • ❌ Responsive design
  • ❌ Adaptation thème
