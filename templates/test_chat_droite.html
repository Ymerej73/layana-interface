🧪 Test du Chat Bot AYORA - Déploiement Droite

📋 Instructions de test

Cette page teste le chat bot qui se déploie depuis le côté droit de l'écran avec une fenêtre plus grande.

  1. Cliquez sur l'icône flottante à droite pour ouvrir le chat
  2. Le chat devrait glisser depuis la droite vers la gauche
  3. L'icône devrait changer en
  4. La fenêtre est maintenant plus grande (450x700px au lieu de 350x500px)
  5. Testez les suggestions rapides et l'envoi de messages

🔧 Tests techniques

📱 Test responsive

Redimensionnez votre navigateur pour tester l'adaptation mobile du chat.

Le chat devrait s'adapter automatiquement à la taille de l'écran.

🎨 Test thème

Changez de thème dans les paramètres pour voir l'adaptation du chat.

Le chat devrait s'adapter automatiquement au thème clair/sombre.
